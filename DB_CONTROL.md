# Control de Versiones de la Base de Datos (SGP-FAMAC)

Este documento es para llevar el control de los cambios y actualizaciones de la base de datos del proyecto.

**Reglas de Uso:** 
1. Si la versión de la base de datos es diferente a la que tenías, anota aquí qué versión subiste, la fecha y tu nombre.
2. **Importante:** Siempre que exista una nueva versión, debes unir/sincronizar tu base de datos local con los nuevos cambios antes de hacer nuevos desarrollos y volver a subir una nueva versión para no perdernos en el desarrollo ni sobreescribir datos importantes.
